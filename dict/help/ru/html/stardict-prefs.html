4.1.1 Сканирование выделенного

Сканировать, только если нажата клавиша-модификатор

Сканировать только при нажатии клавиши-модификатора. Модификатор задаётся параметром Клавиша-модификатор сканирования.

По умолчанию: отключено.

Скрывать всплывающее окно, когда клавиша-модификатор отпущена

Если этот параметр включён, всплывающее окно скрывается, когда пользователь отпускает клавишу-модификатор. Примечание: окно скрывается только в том случае, если указатель мыши находится в том же положении, что и во время показа окна. Программа не только скрывает окно, но и забывает последний выделенный текст; это позволяет повторить его поиск путём нажатия клавиши-модификатора. Если этот параметр отключён, отпускание клавиши-модификатора не влияет на всплывающее окно.

Этот параметр работает только в том случае, если установлен флажок Сканировать, только если нажата клавиша-модификатор .

По умолчанию: включено.

Клавиша-модификатор сканирования

Этот параметр позволяет выбрать клавишу-модификатор или комбинацию клавиш для включения сканирования.

Этот параметр работает только в том случае, если установлен флажок Сканировать, только если нажата клавиша-модификатор .

Клавиша по умолчанию: Win (зависит от версии StarDict).

Сканировать содержимое буфера обмена

Включает сканирование содержимого буфера обмена. Если флажок установлен, при копировании нового текста в буфер обмена StarDict будет выполнять поиск этого текста во всплывающем окне.

Этот параметр доступен только в Windows-версии StarDict.

По умолчанию: отключено.

Комбинация клавиш режима сканирования

Комбинация клавиш режима сканирования включает или отключает этот режим. Для её использования необходимо установить флажок Использовать комбинацию клавиш для включения или отключения сканирования. Её можно изменить в поле, расположенном справа от названия флажка. Щёлкните в пределах поля и нажмите ту комбинацию клавиш, которую следует назначить.

По умолчанию: включено.

Комбинация клавиш по умолчанию: Ctrl+Alt+x.

4.1.2 Шрифт

Использовать свой шрифт

Позволяет переопределить системный шрифт по умолчанию для StarDict. Если этот параметр включён, вместо системного шрифта будет использоваться шрифт, заданный с помощью параметра Шрифт словаря.

Параметры шрифта применяются сразу же.

По умолчанию: отключено.

4.1.3 Кэш

Создать файлы кэша для ускорения загрузки.

Когда словарь отрывается в первый раз, StarDict выполняет некоторые вычисления для ускорения доступа к его содержимому. StarDict может сохранить результаты в специальных файлах кэша, которые будут использоваться при следующем запуске StarDict. Благодаря файлам кэша запуск StarDict выполняется быстрее, так как программе не нужно снова выполнять те же самые вычисления. Настоятельно рекомендуется включить функцию кэширования. Сначала StarDict попытается сохранить файл кэша вместе со словарём. Если это не получится сделать (например, из-за того, что каталог защищён от записи), StarDict создаст файл в домашнем каталоге пользователя, в каталоге StarDict.

По умолчанию: включено.

Сортировать список слов с помощью функции сравнения

Обычно StarDict использует индекс слов для поиска слова в словаре. Когда слово имеется в индексе, показывается соответствующая статья. В ином случае проверяются соседние слова, они могут быть использованы в качестве результата. StarDict использует эвристические алгоритмы для принятия решения о том, можно ли предложить определённое слово в качестве результата поиска. Порядок слов в индексе очень важен; полезна упорядоченность элементов индекса по алфавиту или в соответствии с другими правилами. Функции сравнения позволяют указать ряд предустановленных порядков сортировки, что может быть полезным при работе с определёнными языками.

Чтобы использовать функцию сравнения, установите флажок Сортировать список слов с помощью функции сравнения и выберите нужную функцию в списке Функция сравнения. Функция сравнения будет применена сразу после закрытия диалогового окна настройки. Будьте готовы к тому, что на применение новой функции сравнения может потребоваться достаточно много времени.

Настоятельно рекомендуется включить параметр Создать файлы кэша для ускорения загрузки., чтобы StarDict не приходилось выполнять сортировку при каждом запуске программы.

По умолчанию: отключено.

Очистить кэш

Удалить все файлы кэша StarDict.

4.1.4 Экспорт

StarDict может экспортировать результаты поиска в файл. Есть два варианта: экспортировать только найденное слово или экспортировать результат поиска целиком (т.е. всё содержимое области определений).

Экспортировать только слова

Если этот параметр включён, StarDict выполняет экспорт только найденного слова, в ином случае — экспорт всей статьи.

По умолчанию: включено.

Имя файла

Имя файла, в который следует экспортировать данные.

4.1.5 Звук

В ряде случаев StarDict требуется воспроизводить звуковые файлы. StarDict может сопровождать звуками действия в графическом интерфейсе пользователя. StarDict может произносить слова, поиск которых выполняется в словаре. Некоторые статьи могут содержать ссылки на звуковые файлы — их можно воспроизводить.

Для воспроизведения звуковых файлов StarDict может задействовать заданные платформой службы или же использовать команды, которые указаны пользователем на этой странице.

На этой странице можно настроить некоторые параметры подсистемы синтеза речи. Она используется для произнесения слов компьютером.

Использовать звуки для событий

Следует ли включить звуки для событий. StarDict будет воспроизводить звуки при использовании кнопок и появлении меню.

По умолчанию: включено.

Команда для воспроизведения звуковых файлов

Указывает команду для воспроизведения звуковых файлов. Можно указать имя программы (с полным путём или без него) и любые необходимые параметры. Имя файла для воспроизведения будет добавлено к этой команде во время выполнения.

Всегда использовать команду воспроизведения звуков

На некоторых платформах StarDict сначала задействует заданные ими службы для воспроизведения звуковых файлов, а затем (если службе не удалось воспроизвести файл) использует предоставленную пользователем команду. Установка этого флажка позволяет указать программе, что следует всегда использовать пользовательскую команду.

По умолчанию: отключено.

Каталоги со звуковыми файлами произношений слов:

Позволяет указать каталоги с наборами файлов произношений слов. Эти звуковые файлы содержат образцы речи, начитанные реальными людьми. Обычно в одном файле записано одно слово, при этом название файла совпадает с ним. Звуковые файлы распределены по каталогам. В поле ввода необходимо указать путь к каталогу верхнего уровня. Порядок указания путей в поле ввода имеет значение. StarDict проверяет каталоги сверху вниз, используется первый найденный файл.

По умолчанию
В версии StarDict для Windows

C:\Program Files\WyabdcRealPeopleTTS, C:\Program Files\OtdRealPeopleTTS, WyabdcRealPeopleTTS, OtdRealPeopleTTS

В других версиях StarDict

/usr/share/WyabdcRealPeopleTTS, /usr/share/OtdRealPeopleTTS

Использовать программу синтеза речи

Разрешает использовать указанную ниже команду для чтения слов. Чтобы изменения вступили в силу, необходимо перезапустить StarDict.

По умолчанию: отключено.

Команда

В этом поле можно указать команду, которую следует использовать для произнесения слов. Необходимо предоставить строку, где «%s» указывает место для вставки произносимого слова (или фразы). Чтобы изменения вступили в силу, необходимо перезапустить StarDict. Не забудьте установить флажок выше, если требуется использовать эту команду.

По умолчанию: ничего.

4.1.6 Видео

Команда для воспроизведения видеофайлов

Указывает команду для воспроизведения видеофайлов. Можно указать имя программы (с полным путём или без него) и любые необходимые параметры. Имя файла для воспроизведения можно добавить к команде во время выполнения. Видеофайлы могут быть связаны со словарными статьями.

4.1.7 Отображение словарной статьи

Выделять поисковый запрос

Следует ли выделять поисковый запрос в области определений. Поисковый запрос — это слово, которое было найдено в словарях, заглавное слово статьи.

По умолчанию: отключено.

4.1.8 Dict management

Add new dictionaries to active dictionary group

At start time StarDict scans known directories for dictionary files. This options specifies what to do with new dictionaries. When this option is on, new dictionaries are added to the active dictionary group. Otherwise you have to add new dictionaries to a group manually.

Disabling this option may be useful if you have multiple groups and you want to control precisely what each group contains. Often the last active dictionary group is not appropriate for new dictionaries.

По умолчанию: включено.

Add new plug-ins to active dictionary group

At start time StarDict scans known directories for plug-in files. This options specifies what to do with new plug-ins (belonging to Virtual Dictionary and Network dictionary plug-in categories). When this option is on, new plug-ins are added to the active dictionary group. Otherwise you have to add new plug-ins to a group manually.

Disabling this option may be useful if you have multiple groups and you want to control precisely what each group contains. Often the last active dictionary group is not appropriate for new plug-ins.

По умолчанию: включено.

4.2.1 Сетевые словари

StarDict может искать слова в сетевых словарях с помощью настраиваемого сетевого протокола StarDict. На этой странице настройки можно зарегистрировать учётную запись, выбрать сервер для использования и указать другие связанные параметры.

At the present time StarDict may be configured to query only one server. When network dictionaries are enabled, each time you ask StarDict to lookup a word or phrase, the application sends a request to the configured server. Note that network dictionaries are enabled globally, they are used in both Main and Floating windows, they do not depend on the dictionary group selected.

To configure the list of dictionaries on the server, use the Manage Dictionaries dialog, Network dictionaries tab.

You need to register on the server to use this feature. Click Register an account button on this page to register an account.

Enabling network dictionaries poses a security risk. Your text will be sent to remote servers. Additionally, since network requests do not use any encryption, anyone able to sniff communication can see that text.

Включить сетевые словари

Enables network dictionaries.

По умолчанию: отключено.

4.2.2 Веб-браузер

StarDict может потребоваться открыть веб-ссылку, чтобы начать поиск в Интернете, показать веб-сайт проекта StarDict, показать веб-страницу для загрузки новых словарей или новой версии приложения и так далее. На этой странице настройки можно указать способ открытия веб-ссылки.

Команда для открытия URL-адресов

С помощью этого параметра можно указать команду для открытия URL-адресов. Некоторые версии StarDict по умолчанию задействуют заданные платформами службы для открытия URL-адресов. Чтобы использовать пользовательскую команду, необходимо установить расположенный ниже флажок. В других версиях StarDict открыть веб-ссылку можно только с помощью пользовательской команды.

Можно указать имя программы (с полным путём или без него) и любые необходимые параметры. URL-адрес для открытия будет добавлен к этой команде во время выполнения.

По умолчанию: firefox (зависит от версии).

Всегда использовать эту команду для открытия URL-адресов

Этот параметр доступен только в тех версиях StarDict, которые поддерживают другие способы открытия веб-ссылки.

Установите этот флажок, чтобы использовать для открытия веб-ссылок пользовательскую команду, а не заданные платформами службы.

По умолчанию: отключено.

4.3.1 Ввод

Искать во время набора

When this option is checked, StarDict search for the word you are typing in the search field automatically. StarDict monitors key-press events, it starts searching the word entered as soon as there is no activity for more than a specified amount of time. The time span can be changed in the field below. If the time span is 0, search is started immediately after you change the search field.

Если этот флажок не установлен, потребуется нажать клавишу Ввод, чтобы программа StarDict начала поиск слова.

По умолчанию: включено.

Время ожидания после изменения слова

Время ожидания после изменения слова в миллисекундах. Подробные сведения доступны в приведённом выше описании параметра.

По умолчанию: 320 мс

Показывать первое слово, когда ничего не найдено

Когда StarDict не удаётся найти точное соответствие запрошенному слову, приложение может предложить другое слово, близкое к исходному. Если этот флажок установлен, самое близкое слово будет показано в области определений. В ином случае StarDict сообщит о том, что исходное слово не найдено. Обратите внимание, этот параметр применяется только к простым запросам (тем, где не используются регулярные выражения, шаблоны, нечёткие совпадения и т.д.).

По умолчанию: включено.

4.3.2 Параметры

Стиль

Позволяет настроить графический вид приложения. Выберите пункт По умолчанию для восстановления исходного внешнего вида StarDict (без применения стиля). Чтобы изменения вступили в силу, необходимо перезапустить приложение. Инструкции по подготовке пользовательского стиля доступны в исходных файлах StarDict (файл doc/Skins).

По умолчанию: По умолчанию.

Скрывать главное окно при запуске StarDict

Управляет тем, появляется ли главное окно при запуске программы. Если этот флажок установлен, после запуска StarDict будет показан только значок в области уведомлений. Главное окно можно открыть, щёлкнув по нему.

По умолчанию: отключено.

Использовать комбинацию клавиш для открытия главного окна

Этот параметр позволяет включить комбинацию клавиш, которая открывает главное окно, а также изменить её. Если флажок установлен, при нажатии этой комбинации клавиш пользователю будет показано главное окно.

По умолчанию: включено.

Комбинация клавиш по умолчанию: Ctrl+Alt+z

Прозрачность

Позволяет сделать главное окно прозрачным. Перемещайте ползунок для изменения степени прозрачности (эффект виден сразу же). Прозрачность измеряется в процентах, от 0 до 80, где 0 — полная непрозрачность, 80 — почти полная прозрачность.

На X11 этот параметр работает только для экранов X с запущенным композитным диспетчером окон. В Windows он работает всегда.

По умолчанию: 0 — полная непрозрачность.

4.3.3 Поиск в Интернете

StarDict может использовать поисковые веб-системы, веб-словари, базы данных с веб-интерфейсом и другие веб-сайты, предоставляющие информацию по запросу. Текст для поиска включается в URL-адрес. Обычно текст назначается параметру запроса GET. Например, для поиска слова stardict в поисковой системе Google можно использовать следующий URL-адресhttp://www.google.com/search?q=stardict. В StarDict можно выполнять поиск на веб-сайтах, щёлкнув правой кнопкой мыши по кнопке Поиск в Интернете в главном окне.

Приложение StarDict не обрабатывает веб-запрос само, а только составляет URL-адрес и вызывает внешнюю программу (например, веб-браузер) или использует заданные платформой службы для открытия этого URL-адреса.

На этой странице представлен список веб-сайтов. Кнопки Добавить и Удалить позволяют добавлять новые сайты и удалять существующие записи. С помощью кнопок со стрелками Вверх и Вниз можно изменить порядок записей в списке. Каждая запись состоит из трёх элементов:

Запись веб-сайта
Название сайта

Это подпись, которая показана в меню. Используйте значимое имя.

Адрес сайта

Ссылка на главную страницу веб-сайта. Эта ссылка открывается, когда нечего искать (поле поиска пусто).

Поисковая ссылка

Ссылка для выполнения запроса. %s указывает, где вставляется текст для поиска. Например: http://www.google.com/search?q=%s.

4.4 Значок в области уведомлений

  • 4.4.1. Параметры

4.4.1 Параметры

Когда нажата средняя кнопка мыши

Указывает действие, которое следует выполнить, когда пользователь щёлкает средней кнопкой мыши по значку в области уведомлений. Доступны следующие варианты:

  • Искать во всплывающем окне
  • Искать в главном окне
  • Ничего не делать

Если выбран вариант Ничего не делать, StarDict просто игнорирует щелчки средней кнопкой мыши. В ином случае StarDict получает текст текущего выделения, выполняет его поиск и показывает результат либо во всплывающем окне, либо в главном окне.

Если текст выделен в нескольких окнах на рабочем столе, может быть непонятно, какой текст будет использован. StarDict работает с основным выделением X. Выделением одновременно может владеть только одно приложение, поэтому при выборе пользователем какого-либо текста в приложении это приложение становится владельцем выделения; предыдущий владелец утрачивает владение выделением. Вернёмся к рассматриваемому вопросу. StarDict будет использовать тот текст, который был выделен последним.

По умолчанию: Искать во всплывающем окне.

4.5 Всплывающее окно

  • 4.5.1. Параметры
  • 4.5.2. Параметры

4.5.1 Параметры

Произнести слово при появлении всплывающего окна

Если этот параметр включён, слово будет произноситься при показе перевода во всплывающем окне.

По умолчанию: отключено.

Показать всплывающее окно, если слово не найдено

Этот параметр позволяет настроить действие, которое следует выполнять, когда не удалось найти слово для показа во всплывающем окне. Доступны два варианта: 1) вообще не показывать всплывающее окно (для этого нужно снять флажок), 2) показать всплывающее окно с сообщением о том, что слово не было найдено (для этого нужно установить флажок).

По умолчанию: включено.

4.5.2 Параметры

Максимальная ширина окна

Устанавливает максимальную ширину всплывающего окна.

По умолчанию: 320.

Максимальная высота окна

Устанавливает максимальную высоту всплывающего окна.

По умолчанию: 240.

Использовать свой цвет фона

Позволяет использовать настраиваемый цвет фона во всплывающем окне и выбрать этот цвет.

По умолчанию: отключено.

Прозрачность

Позволяет сделать всплывающее окно прозрачным. Перемещайте ползунок для изменения степени прозрачности (эффект виден сразу же). Прозрачность измеряется в процентах, от 0 до 80, где 0 — полная непрозрачность, 80 — почти полная прозрачность.

На X11 этот параметр работает только для экранов X с запущенным композитным диспетчером окон. В Windows он работает всегда.

По умолчанию: 0 — полная непрозрачность.
