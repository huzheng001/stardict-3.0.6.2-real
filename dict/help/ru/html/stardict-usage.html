3.1 Поиск слова

Для поиска перевода слова следует выполнить следующие шаги:

  1. Введите слово, которое требуется найти, в поле поиска.
  2. StarDict немедленно отобразит перевод. Если перевод не найден, можно нажать клавишу Return для выполнения нечёткого запроса или просто нажать кнопку Нечёткий запрос.
  3. Воспользуйтесь полосой прокрутки для перемещения по переводу. Можно выбирать слова в списке слов.

3.2 Очистка области отображения

Чтобы очистить поле поиска, нажмите кнопку Очистить, расположенную слева, или просто нажмите Alt-C или ESC.

3.3 Копирование текста в буфер обмена

Есть два способа скопировать текст в области определений в другое приложение:

  • Чтобы скопировать весь текст, нажмите кнопку Копировать, расположенную над Областью определений.
  • Чтобы скопировать часть текста, выделите её, нажмите правую кнопку мыши и выберите в контекстном меню пункт Копировать.

3.4 Поиск текста в области определений

Для поиска текста в области определений следует выполнить следующие шаги:

  1. Нажмите кнопку Найти, расположенную над Областью определений.
  2. Введите текст, который следует найти, в текстовом поле Найти и нажмите клавишу Return. Использовать подстановочные знаки нельзя.
  3. Для поиска следующего совпадения нажмите клавишу Return снова.

3.5 Сканирование: руководство по эксплуатации

  • В приведённых далее сценариях предполагается, что выделение выполняется с помощью мыши.
  • Клавиша сканирования — это клавиша-модификатор, заданная с помощью параметра «Клавиша-модификатор сканирования», см. Раздел 4.1.1 ― Сканирование выделенного.
  • 3.5.1. Сканирование в Linux и Unix
  • 3.5.2. Сканирование в Windows

3.5.1 Сканирование в Linux и Unix

Случай 1. Параметр «Сканировать, только если нажата клавиша-модификатор» отключён
  1. Выделите текст для перевода.
  2. Подождите несколько секунд. Появится всплывающее окно.
Случай 2. Параметр «Сканировать, только если нажата клавиша-модификатор» включён
  1. Нажмите и удерживайте клавишу сканирования.
  2. Выделите текст.
  3. Подождите несколько секунд. Появится всплывающее окно.
  4. Отпустите клавишу сканирования.
Случай 2.1. Параметр «Сканировать, только если нажата клавиша-модификатор» включён. Вариант
  1. Выделите текст и не отпускайте кнопку мыши.
  2. Нажмите и удерживайте клавишу сканирования.
  3. Отпустите кнопку мыши.
  4. Подождите несколько секунд. Появится всплывающее окно.
  5. Отпустите клавишу сканирования.

Выделение выполняется при отпускании кнопки мыши. В это время необходимо нажать клавишу сканирования. StarDict требуется некоторое время на анализ выделенного фрагмента, поэтому всё это время (несколько секунд) необходимо удерживать клавишу сканирования нажатой.

3.5.2 Сканирование в Windows

Случай 1. Параметр «Сканировать, только если нажата клавиша-модификатор» отключён
  1. Наведите указатель мыши на слово, которое требуется перевести.
  2. Не перемещайте мышь в течение нескольких секунд. Появится всплывающее окно.
Случай 2. Параметр «Сканировать, только если нажата клавиша-модификатор» включён
  1. Наведите указатель мыши на слово, которое требуется перевести.
  2. Нажмите и удерживайте клавишу сканирования.
  3. Не перемещайте мышь в течение нескольких секунд. Появится всплывающее окно.
  4. Отпустите клавишу сканирования.

Информация о сканировании в PDF-документах: Раздел 2.6 ― Модуль WordPick Adobe Acrobat.

3.6 Включение или отключение режима сканирования

Существует ряд способов включения или отключения режима сканирования выделенного. В режиме сканирования StarDict показывает перевод текста, выделяемого пользователем в различных окнах. См. Раздел 2.5 ― Сканирование выделенного.

Способы включения режима сканирования
  • Установите флажок Сканирование выделенного в нижнем левом углу главного окна StarDict.
  • Щёлкните правой кнопкой мыши по значку в области уведомлений и выберите в появившемся контекстном меню пункт Сканировать.
  • Нажмите комбинацию клавиш Ctrl+Alt+x (она включает или отключает режим сканирования). Для настройки комбинации клавиш см. Раздел 4.1.1 ― Сканирование выделенного.

3.7 Комбинации клавиш главного окна

Ctrl+q

Выйти из StarDict.

Alt+x

Свернуть в лоток. Главное окно StarDict закрывается, элемент на панели задач исчезает, но приложение продолжает работать. Главное окно можно открыть снова, щёлкнув по значку в области уведомлений.

Alt+z

Свернуть приложение. Главное окно скрывается, элемент на панели задач остаётся.

Alt+e

Сохранить определение слова в файл. Дополнительная информация: Раздел 2.3 ― Панель инструментов области определений.

F1

Показать справку.

Ctrl+f

Поиск в области определений. Дополнительная информация: Раздел 2.3 ― Панель инструментов области определений.

Alt+Стрелка «влево»

Назад. Возврат к слову, поиск которого был выполнен ранее.

Alt+Стрелка «вверх»

Предыдущее слово. Поиск предыдущего слова в индексе слов.

Alt+Стрелка «вниз»

Следующее слово. Поиск следующего слова в индексе слов.

Alt+m

Меню. Показать главное меню.

Ctrl+v

Поиск текста из буфера обмена в StarDict.

Ctrl+u

Очистить поле поиска и перейти в него.

BackSpace

Аналогично Ctrl+u.

Ввод

  1. Переход в список слов.

    Если у выбранного элемента индекса есть дочерние элементы, он будет развёрнут или свёрнут. Если у выбранного элемента индекса нет дочерних элементов, поиск этого элемента будет выполнен в главном окне.

  2. Переход в область определений.

    Повторный поиск текста в поле поиска.

Пробел

Переход в поле поиска.

Escape

  1. Переход на панель поиска в области определений.

    Скрыть панель поиска.

  2. В ином случае эта комбинация клавиш будет работать так же, как Ctrl+u.

Ctrl+Alt+z

Открыть главное окно StarDict. Главное окно может быть свёрнуто, или же StarDict может существовать только в виде значка в области уведомлений. Нажатие этой комбинации клавиш в любом случае приведёт к тому, что пользователю будет показано главное окно. Её можно изменить в диалоге настройки, см. Раздел 4.3.2 ― Параметры.

Ctrl+Alt+x

Переключить режим сканирования. Эту комбинацию клавиш можно изменить в диалоге настройки, см. Раздел 4.1.1 ― Сканирование выделенного.

3.8 Installing new dictionaries

Dictionaries for StarDict you download are normally packed in archives. You must unpack them and copy into one of the StarDict dictionary directories. If you simply copy an archive file into the dictionary directory, StarDict will not find the dictionary. It is recommended to place each dictionary in its own directory. You may build a hierarchy of directories to organize your dictionary collection. StarDict search directories recursively. See also Раздел 7.7 ― StarDict dictionaries.

Installing a dictionary

  1. Download a dictionary. Normally a dictionary is packed in an archive file, for example, WineEnRu.zip.

  2. Unpack the archive in a new directory. You may name the directory after the archive name, though that is not required. 

    WineEnRu/WineEnRu.dict
WineEnRu/WineEnRu.idx
WineEnRu/WineEnRu.ifo

  3. Copy the dictionary directory into one of StarDict dictionary directories, see Раздел 7.7.1 ― Directories.

  4. Restart StarDict. The new dictionary will be added into the default dictionary group.

Broken dictionaries

Some dictionaries you may find on Internet violate restrictions of StarDict dictionary file format. Using such a dictionary may result in StarDict crash. If StarDict becomes unstable, check your dictionaries, especially new ones, with StarDict Editor or stardict-verify command line tool.

3.9 Removing dictionaries

To remove a dictionary it's enough to remove all files belonging to that dictionary. All files of a dictionary share the same base name, for example. 

WineEnRu.dict
WineEnRu.idx
WineEnRu.idx.oft
WineEnRu.ifo

Some dictionaries may include resource database that is named differently. For complete list of file see Раздел 7.7 ― StarDict dictionaries. If you follow the rule "one dictionary ‒ one directory", removing a dictionary will be easer.

Removing a dictionary
  1. Quit StarDict. Make sure that StarDict has really terminated, not just minimized into tray icon!
  2. Remove dictionary files.
