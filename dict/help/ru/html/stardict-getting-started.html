2.1 Запуск StarDict

StarDict можно запустить следующим образом:

Меню Приложения

Выберите пункт меню Инструменты ▸ StarDict.

Командная строка

Введите команду stardict, затем нажмите клавишу Return.

Можно указать также слово для поиска:

stardict слово

или шаблон для поиска:

stardict сло?о*

или выполнить нечёткий запрос:

stardict /слово

Если приложение StarDict уже запущено. В версии для Windows StarDict активирует первый экземпляр StarDict, затем завершает работу. В версии для Gnome StarDict активирует первый экземпляр StarDict, выполняет поиск слова в этом экземпляре (если оно было указано в командной строке), затем завершает работу.

Полное описание параметров командной строки: Раздел 7.1 ― Параметры командной строки.

2.2 Главное окно StarDict

Когда вы запускаете StarDict, появляется следующее окно.

Рисунок 1 Главное окно StarDict

Главное окно StarDict состоит из следующих элементов:

Верхняя панель

Верхняя панель содержит Поле поиска и несколько кнопок. Далее они перечислены в порядке слева направо.

Очистить

Очищает поле поиска.

Поле поиска

Используйте поле поиска для ввода слов, которые требуется найти. В раскрывающемся списке показаны последние запрошенные слова.

Нечёткий запрос

Выполняет нечёткий запрос слова, введённого в поле поиска.

Назад

Вернуться к слову, запрошенному ранее. Щёлкните правой кнопкой мыши для просмотра списка запрошенных ранее слов.

Главное меню

Кнопка в правом углу верхней панели. Она открывает главное меню StarDict.

Левая панель

Левая панель содержит несколько кнопок, далее они перечислены в порядке сверху вниз:

Список слов

Открывает список слов, которые были найдены при поиске.

Список результатов

Открывает список источников (главным образом — словарей), где найдено текущее слово. В области определений находятся результаты из всех источников. Если щёлкнуть по какому-либо источнику в списке, в окне области выполняется прокрутка к определению из этого источника.

Полнотекстовый перевод

Переключает в режим полнотекстового перевода, который позволяет переводить блоки текста. См. Раздел 2.7 ― Полнотекстовый перевод.

Группы словарей

Shows a pop-up menu that allows to select a group of dictionaries where to search words. The menu also contains a checkbox to enable network dictionaries. See Раздел 4.2.1 ― Сетевые словари for information about network dictionaries and how to configure them.

Предыдущее слово и Следующее слово

Эти кнопки расположены в нижней части левой панели. Они используются для работы со списком слов и позволяют переключаться между предыдущим и следующим словами в нём.

Нижняя панель

Левая часть панели в порядке слева направо:

Флажок Сканирование выделенного

Включает или отключает режим поиска выделенного. Если флажок Сканирование выделенного установлен, StarDict будет автоматически пытаться найти слова, выделенные пользователем в любом приложении. Дополнительные сведения о режиме поиска выделенного: Раздел 2.5 ― Сканирование выделенного.

Информация

Показывает вводную информацию об использовании StarDict в области определений.

Выйти

Закрывает StarDict.

Правая часть панели в порядке слева направо:

Поиск в Интернете

Нажмите кнопку для выполнения поиска в Интернете. По умолчанию используется первый веб-сайт, но по щелчку правой кнопкой мыши можно выбрать любой настроенный веб-сайт. StarDict выполняет поиск слова, указанного в поле поиска, путём отправки запроса GET к настроенной веб-странице. Результаты отображаются в веб-браузере. Настройка

Открывает диалоговое окно Настройка, которое позволяет настроить множество параметров StarDict. См. Раздел 4 ― Настройка.

Список слов

Меньшая половина главного окна StarDict, расположенная слева. Она содержит список слов, которые были найдены при поиске. Щёлкните по слову для просмотра связанных с ним статей в области определений. Для разделения пространства между списком слов и областью определений можно воспользоваться специальным разделителем в середине главного окна StarDict.

Для разделения пространства между списком слов и областью определений можно воспользоваться специальным разделителем в середине главного окна StarDict. Область определений содержит свои собственные панели: панель инструментов сверху (см. Искать выбранный в области определений текст в главном окне. В версии StarDict для Windows в том случае, если текст не выбран, в главном окне будет выполнен поиск содержимого буфера обмена.

правая кнопка мыши

Показать всплывающее меню.

Панель инструментов области определений находится в её верхней части. Она содержит несколько кнопок. Далее они перечислены в порядке слева направо.

Скрыть или показать список слов

Скрыть или показать список слов или список результатов (в зависимости от того, какой из них активен)

Копировать

Копировать весь текст в области определений в буфер обмена.

Произнести слово

Это кнопка с дополнительным раскрывающимся меню. При её нажатии StarDict произнесёт текущее слово с помощью первой программы синтеза речи, которая позволяет это сделать. В раскрывающемся меню можно выбрать программу, которую следует использовать.

Программы синтеза речи
Синтез речи «Реальные люди»

Эта программа использует звуковые образцы, записанные реальными людьми.

Программа синтеза речи Espeak

Модуль синтеза речи Espeak.

Программа синтеза речи Festival

Модуль синтеза речи Festival.

Команда запуска синтеза речи

Команда запуска синтеза речи, которую можно указать в меню «Настройка». См. Раздел 4.1.5 ― Звук.

В зависимости от параметров настройки, установленных модулей и прочих факторов, перечень программ синтеза речи может быть более коротким.

Сохранить в файл

Сохранить текст, расположенный в области определений, в файл. Новый текст добавляется в конец файла. Файл для использования указывается в разделе Раздел 4.1.4 ― Экспорт диалога Настройка. Также можно задать экспорт только найденного слова, а не его определения. StarDict выполняет экспорт не того слова, которое было указано в поисковом запросе, а того слова, которое было найдено (т.е. слова, перевод которого показан в области определений). Если флажок «Экспортировать только слова» не установлен, учитывается выделение. Когда в области определений выделяется текст, он экспортируется (при этом найденное слово предваряет этот текст). Если же текст не был выделен, экспортируется весь текст в области определений.

Комбинация клавиш: Alt+e

Искать в области определений слова

Искать текст в области определений. Эта кнопка открывает в нижней части области определений панель поиска, где можно указать слово для поиска.

Комбинация клавиш: Ctrl+f

2.4 Значок в области уведомлений

StarDict также добавляет значок в область уведомлений. При сворачивании главное окно скрывается. После щелчка по этому значку окно будет показано снова. Если щёлкнуть по значку правой кнопкой мыши, появится всплывающее меню, которое позволяет включить или отключить опциюСканирование выделенного или воспользоваться опцией Выйти. Включить или отключить опцию Сканирование выделенного также можно напрямую, путём щелчка по значку при нажатой клавише Ctrl. Если щёлкнуть по значку с нажатием средней кнопки мыши, будет выполнен поиск выделенного слова.

2.5 Сканирование выделенного

StarDict может автоматически выполнять сканирование выделенного: когда слово или фраза выделяются в StarDict или в любом другом приложении, StarDict получает выделенный текст и выполняет его поиск, а затем показывает всплывающее окно с результатом.

В Windows режим сканирования работает иначе. Он никак не связан с выделением. StarDict извлекает расположенный под указателем мыши текст, когда перемещение мыши на время приостанавливается. Обычно StarDict выполняет поиск слова, находящегося под указателем. Если указатель расположен не над словом, выполняется поиск находящихся слева и справа от указателя слов. Стоит уточнить, что именно StarDict ищет как слово. StarDict выполняет ряд попыток. Сначала программа рассматривает слово, содержащее любые символы, кроме пробелов и знаков препинания; затем — любое слово, состоящее только из букв; слово из букв в нижнем регистре; слово из букв в верхнем регистре; слово из букв, начинающееся с прописной. В итоге StarDict использует одно из рассмотренных слов для нахождения самой длинной словообразующей приставки. Можно помочь StarDict извлечь правильное слово или часть слова, выполнив выделение. Хотя StarDict не работает с выделенными фрагментами, такое выделение может повлиять на результаты поиска с помощью StarDict.

В Windows сканирование может не работать в некоторых приложениях. Например, известно, что сканирование не работает в FireFox 3.x и Windows Console (процесс cmd.exe). Функция сканирования зависит от того, каким образом приложение выводит текст. Если сканирование не работает, можно воспользоваться другой возможностью StarDict — Сканировать содержимое буфера обмена, см. Раздел 4.1.1 ― Сканирование выделенного.

В Windows StarDict особым образом работает с несколькими приложениями. Во-первых, сама программа StarDict. Чтобы выполнить сканирование слова в главном окне StarDict, необходимо выбрать его. Режим сканирования в главном окне StarDict работает точно так, как он должен это делать в общем случае; то, что работа ведётся в Windows, не играет роли. Тем не менее, в других окнах StarDict выполнять сканирование нельзя.

Во-вторых, имеется специальный подключаемый модуль для сканирования в PDF-документах, см. Раздел 2.6 ― Модуль WordPick Adobe Acrobat.

Рисунок 3 Всплывающее окно StarDict
Советы по использованию
  • Всплывающее окно можно переместить с помощью перетаскивания.
  • Всплывающее окно исчезает после перемещения указателя мыши на некоторое расстояние от окна.
  • StarDict можно настроить так, чтобы программы выполняла сканирование только при нажатии специальной клавиши-модификатора. См. Раздел 4.1.1 ― Сканирование выделенного.
  • Режим сканирования можно включать и отключать путём нажатия комбинации клавиш Ctrl+Alt+x. См. Раздел 4.1.1 ― Сканирование выделенного.
  • StarDict запоминает последний текст, поиск которого выполнялся во всплывающем окне, и не ищет его снова. Чтобы просмотреть перевод ещё раз, следует сначала выполнить поиск другого слова (недостаточно просто изменить выделение), а затем выполнить поиск исходного слова. Другой вариант — щёлкнуть средней кнопкой мыши по значку в области уведомлений; StarDict выполнит поиск выделенного текста без каких-либо дополнительных условий.

Всплывающее окно StarDict содержит следующие элементы:

Панель кнопок инструментов

Панель, расположенная в верхней части всплывающего окна. Она автоматически скрывается в целях экономии пространства для отображения. Когда указатель мыши наводится на всплывающее окно, панель кнопок инструментов немедленно появляется, а при прекращении наведения — скрывается вновь.

Эти кнопки могут использоваться для поиска слова в главном окне, копирования результата поиска в буфер обмена, произнесения слова, отключения сканирования выделенного и так далее. Самая правая кнопка — Закрепить всплывающее окно. Если всплывающее окно закреплено, оно всегда будет отображаться в этом месте.

Область определений

Результаты поиска отображаются в этой области.

2.6 Модуль WordPick Adobe Acrobat

StarDict включает модуль Abode Acrobat, который можно использовать для извлечения и перевода текста из документов в формате PDF. Модуль WordPick доступен только в версии StarDict для Windows. Основной механизм извлечения текста не работает с PDF-документами в Windows.

Ограничения

Для работы модуля WordPick необходимо установить платное издание Acrobat (Acrobat Stardard, Pro или издание с большим количеством функций); модуль не будет работать с бесплатным продуктом Acrobat Reader. Модуль совместим с версиями 7-9 Acrobat. Параметры безопасности, установленные для PDF-документов, должны разрешать извлечение содержимого этих документов.

Модуль WordPick добавляет на стандартную панель инструментов Acrobat значок (рядом с инструментом «Рука»). В Acrobat Reader этот значок не появится — модуль не поддерживается. Если для текущего документа запрещено извлечение текста, значок будет неактивным; модуль не будет работать.

Рисунок 4 Панель инструментов Adobe Acrobat со значком модуля WordPick
Изучение параметров безопасности документов

Воспользуйтесь главным меню: Файл, Свойства документа. Появится диалоговое окно Свойства документа. Перейдите на вкладку Безопасность. Посмотрите на группу параметров Обзор ограничений документа. Копирование или извлечение содержимого должно быть разрешено.

Перевод текста с помощью модуля WordPick
  1. Выберите инструмент WordPick и инструмент выделения. Курсор превратится в вертикальную полоску. Теперь можно выделить текст с помощью мыши.
  2. Запустите StarDict, если программа ещё не запущена, и включите режим сканирования.
  3. После выделения слова в PDF-документе StarDict покажет всплывающее окно с переводом, если это слово было найдено.

Можно выделять последовательности слов и фраз. Если StarDict не удалось найти указанную фразу, программа попытается убирать слова, начиная с конца. Если больше ничего не удастся найти, будет показан перевод первого слова.

Следует учитывать, что клавиша-модификатор сканирования (Раздел 4.1.1 ― Сканирование выделенного) не работает в Acrobat. WordPick выполняет поиск текста при его выделении в документе независимо от того, нажата ли клавиша-модификатор. Чтобы отключить режим перевода текста, отмените выбор инструмента WordPick на панели инструментов (отменять выбор инструмента выделения необязательно).

2.7 Полнотекстовый перевод

StarDict позволяет выполнять полнотекстовый перевод. С помощью программы можно переводить короткие тексты (примерно до 2000 символов). Точные цифры зависят от системы перевода и других факторов. Чтобы перейти в режим полнотекстового перевода, нажмите кнопку Полнотекстовый перевод на левой панели в главном окне StarDict.

Для выполнения перевода StarDict использует доступные в Интернете бесплатные онлайн-сервисы. StarDict собирает текст, исходный и целевой языки, создаёт HTTP-запрос и отправляет его онлайн-сервису перевода. Приложение получает ответ сервиса и отображает результат в окне. Ниже показан интерфейс StarDict к сервису полнотекстового перевода.

Рисунок 5 Режим полнотекстового перевода

Верхняя строка окна содержит (слева направо): поле со списком для выбора системы перевода, поле со списком для выбора исходного языка и поле со списком для выбора целевого языка. Под ней располагается область для переводимого текста. Нижняя часть окна содержит область для показа результата перевода. Кнопка Перевести в середине окна отправляет запрос на перевод. В нижней строке окна находится ссылка на систему онлайн-перевода. Она открывается в веб-браузере.

Чтобы выполнить перевод, необходимо сначала выбрать систему перевода. Доступны следующие варианты: Google, Yahoo, Excite Japan. Выберите исходный и целевой языки. Перечень доступных исходных и целевых языков зависит от выбранной системы перевода. Доступные целевые языки также зависят от исходного языка. Введите в области исходного текста тот текст, который требуется перевести. Нажмите кнопку Перевести для начала перевода. Результат перевода будет показан в соответствующей области. Иногда там может быть показан текст Не найдено!, означающий, что перевод выполнить не удалось. Это может произойти по ряду причин: превышение допустимой длины исходного текста, неработоспособность онлайн-сервиса, смена формата вывода сервисом (из-за чего StarDict не удалось выполнить синтаксический разбор). В области результата могут быть показаны и другие ошибки.
